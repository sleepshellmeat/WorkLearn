什么是区块链？
# 是使用密码学算法产生的区块，用时间链在一起，得名区块链。
详细信息
# 区块就是存储了交易的头信息和交易的列表信息的数据结构
# Block Header(区块头)
# Transaction list(交易列表)里面存储了一笔一笔的交易信息
# prev(上一个)这个一般是一个区块上一个区块的编号
# Hash（加密算法）一个区块的Hash改变整个链的Hash全部改变
定义
# 因为区块与区块直接相连，世界上的节点全部都要同步，所以：
# 区块链有不可篡改，永不消失，可追溯，的优势。
# 被誉为制造信用的机器。
